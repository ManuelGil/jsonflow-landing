## Features Section

### Key Features

- **Instant Graphs**  
  Explore deeply nested structures with a single click.

- **Seamless VS Code Integration**  
  Works right where you code – inside VS Code. No setup required.

- **Multi-Format Support**  
  JSON, YAML, XML, CSV, and many more.
